BAG Address Lookup API

GET /lookup

Look up a public space and locality by postal code and house number.

ParamDescription
pcPostal code (e.g. 1234AB)
nHouse number

Example: /lookup?pc=1234AB&n=10

GET /suggest

Suggest localities matching a prefix.

ParamDescription
wpLocality name prefix

Example: /suggest?wp=Amst

GET /localities

List all localities. Returns a JSON array of objects:

FieldDescription
wpLocality name
gmMunicipality name
gm_codeMunicipality code

GET /municipalities

List all municipalities. Returns a JSON array of objects:

FieldDescription
gmMunicipality name
gm_codeMunicipality code
pvProvince name
