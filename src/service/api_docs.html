BAG Address Lookup API

GET /lookup

Look up a public space and locality by postal code and house number.

ParamDescription
pcPostal code (e.g. 1234AB)
nHouse number

Example: /lookup?pc=1234AB&n=10

GET /suggest

Suggest localities and municipalities matching a query. Returns a JSON array mixing locality objects (same shape as /localities) and municipality objects (same shape as /municipalities); locality entries carry a wp field while municipality entries do not.

ParamDescription
wpLocality or municipality name (prefix / fuzzy)

Example: /suggest?wp=Amst

GET /localities

List all localities. Returns a JSON array of objects:

FieldDescription
wpLocality name
wp_codeBAG woonplaatsidentificatiecode
gmMunicipality name
gm_codeMunicipality code
pvTwo-letter province code (e.g. NH, ZH)
uniqueTrue if the name is not shared with any other locality or municipality (a locality matching its own municipality's name does not count as a collision)
had_suffixTrue if the source BAG name carried a disambiguating province suffix that was stripped (e.g. Loo Gld → Loo). Always implies unique: false.

GET /municipalities

List all municipalities. Returns a JSON array of objects:

FieldDescription
gmMunicipality name
gm_codeMunicipality code
pvTwo-letter province code (e.g. NH, ZH)
uniqueTrue if the name is not shared with any other municipality or with any locality outside this municipality
had_suffixTrue if the CBS name carried a disambiguating province suffix that was stripped (e.g. Hengelo (O.) → Hengelo). Always implies unique: false.
